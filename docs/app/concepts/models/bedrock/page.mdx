## Configuration

To use AWS Bedrock, set the following environment variables:

```bash
DEFAULT_PROVIDER=bedrock
AWS_REGION=us-east-1
```

<Callout type="warning">
  Regional availability may change. Check the [AWS Bedrock documentation](https://docs.aws.amazon.com/bedrock/latest/userguide/batch-inference-supported.html) for the latest information.
</Callout>

## Example Request

```bash
curl -X POST http://localhost:8000/cargo/load \
  -H "Content-Type: application/json" \
  -H "X-API-Key: convoy_sk_your_key_here" \
  -d '{
    "params": {
      "model": "amazon-nova-pro",
      "max_tokens": 100,
      "messages": [{"role": "user", "content": "Hello"}]
    },
    "callback_url": "https://example.com/callback"
  }'
```
