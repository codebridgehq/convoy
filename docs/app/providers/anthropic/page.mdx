# Anthropic

Use Anthropic's Message Batches API directly.

import { Callout } from 'nextra/components'

## Prerequisites

- Anthropic API key

## Configuration

```bash
# .env
DEFAULT_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-ant-api03-...
```

## Supported Models

Convoy uses provider-agnostic model IDs. When using Anthropic, these are automatically translated to Anthropic model identifiers.

  See [Supported Models](/concepts/models) for the full list of available models.
</Callout>

## Example Request

Use the provider-agnostic model ID in your requests:

```bash
curl -X POST http://localhost:8000/cargo/load \
  -H "Content-Type: application/json" \
  -H "X-API-Key: convoy_sk_your_key_here" \
  -d '{
    "params": {
      "model": "claude-3-haiku",
      "max_tokens": 100,
      "messages": [{"role": "user", "content": "Hello"}]
    },
    "callback_url": "https://example.com/callback"
  }'
```

## How It Works

1. You submit a request with a provider-agnostic model ID (e.g., `claude-3-haiku`)
2. Convoy validates the model and translates it to the Anthropic model identifier
3. Convoy collects requests
4. Submits batch via Anthropic Message Batches API
5. Polls batch status until complete
6. Retrieves results from API
7. Delivers results via callbacks

## API Limits

Anthropic batch API has these limits:

| Limit | Value |
|-------|-------|
| Max requests per batch | 10,000 |
| Max concurrent batches | Varies by tier |
| Batch expiration | 24 hours |

Convoy's default batch size (100) is well within these limits.

## Pricing

Batch API pricing is typically 50% cheaper than real-time API. Check [Anthropic's pricing page](https://www.anthropic.com/pricing) for current rates.
